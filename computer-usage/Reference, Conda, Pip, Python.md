# Conda

- `source activate <env>` Enter environment
- `conda env list`
- `conda --info envs`
- `conda install python=3.5` Change environment's python version
- `conda create -n <name> [python=3.6]`
- `conda i
- `conda install torch=0.4.0 -c pytorch`
