# Linux console emulator

Disallow network access and run emulator.
`unshare -rn mednafen <{zip|rom}file> [-sound 0]`
### config mednafen
Override keys, joystick by creating custom config files at `$HOME/.mednafen`, e.g. `$HOME/.mednafen/nes.cfg`
